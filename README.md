# create-your-data

In this assignment you will

1. Use an rmarkdown file to write the code to access your chosen book from _Project Gutenberg_ usingthe `gutenbergr` package.
2. Create a data folder.
3. Save the file in your data folder.
4. Do a basic tokenization of the data and remove stop words using `tidytext`'s stop_words data.
5. Save the tokenized data.
6. Update this readme file to identify the name of your book.
7. Commit and push your work to Github.

You can use the quakersaints example as a model for how to do this or you can follow the model in the Tidy Text Mining book.
